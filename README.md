\# SwiftPDF Tools



A professional, single-page PDF utility web application hosted on GitHub Pages.



\## Features



\- \*\*Image to PDF Converter\*\*: Convert multiple images into a single PDF file

\- \*\*PDF Compressor\*\*: Reduce PDF file size by re-rendering pages and scaling content

\- \*\*Client-Side Processing\*\*: All operations happen in the browser - no data is sent to servers

\- \*\*Modern UI\*\*: Clean, responsive design built with Tailwind CSS

\- \*\*Drag \& Drop\*\*: Intuitive file uploading with drag and drop support

\- \*\*Mobile Friendly\*\*: Fully responsive design that works on all devices



\## How to Use



1\. \*\*Image to PDF Converter\*\*:

&nbsp;  - Click the "Browse Images" button or drag \& drop images

&nbsp;  - Arrange images using the move up/down buttons

&nbsp;  - Select PDF options (page size, orientation)

&nbsp;  - Click "Convert to PDF" and download your file



2\. \*\*PDF Compressor\*\*:

&nbsp;  - Click the "Browse PDF" button or drag \& drop a PDF file

&nbsp;  - Adjust compression settings (quality level, scale factor)

&nbsp;  - Click "Compress PDF" and download the compressed file



\## Technologies Used



\- HTML5, CSS3, JavaScript (ES6+)

\- \[Tailwind CSS](https://tailwindcss.com/) for styling

\- \[jsPDF](https://github.com/parallax/jsPDF) for PDF generation

\- \[PDF-Lib](https://github.com/Hopding/pdf-lib) for PDF manipulation

\- \[Font Awesome](https://fontawesome.com/) for icons



\## Browser Compatibility



Works on all modern browsers that support:

\- HTML5 File API

\- Canvas API

\- JavaScript ES6+ features



\## Privacy



All processing happens locally in your browser. Your files never leave your device or are sent to any server.



\## Deployment on GitHub Pages



1\. Fork or clone this repository

2\. Push the code to your GitHub repository

3\. Go to repository Settings > Pages

4\. Select the branch (main/master) and folder (root)

5\. Your site will be published at `https://\[username].github.io/\[repository-name]`
